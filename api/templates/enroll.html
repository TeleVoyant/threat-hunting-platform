     Bulk rollout tips
  
  

    

      For 20–100 personal laptops on a flat LAN (no AD, no remote-exec), the practical patterns are:
    

    
    
      
  • One multi-use token, share the link — set Capacity to your endpoint count (or Unlimited for a rollout day), paste the one-liner into a group chat / email, each laptop owner runs it. Revoke the moment the rollout window closes.
    • 
      
  • QR for in-person — print the QR or project on screen; each owner scans on their phone, copies the URL, pastes into elevated PowerShell.
    • 
      
  • One token per endpoint — Capacity = 1 (default in earlier versions). Tightest blast radius if a link leaks. Walk over (or RDP in) to each machine.
    • 
      
  • Standalone bundle.zip — air-gapped seeding via USB. The bundle has the same single/multi-use semantics as the URL.
    • 
      
  • Air-gapped serving — the Wazuh MSI + Sysmon ZIP are cached on the platform; endpoints only reach the platform API, never the public internet.
